TUTORIAL - DEATH CHASE

Controles

Use as setas para controlar o ninja azul

Use as teclas WASD para controlar o ninja vermelho

Objetivo

O jogo é entre dois players e o objetivo do jogo é fugir das shurikens que se movimentam aleatoriamente numa velocidade constante pelo mapa. Caso um dos player encoste na shuriken, ele perderá.

Regras

Os ninjas podem colidir entre si.

Os ninjas podem percorrer todo o mapa, de canto a canto.

Quando um dos players perderem, pressione F5 para reiniciar a partida.
